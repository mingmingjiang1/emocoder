debug

日志工具

开发者使用：
trace：打印调用堆栈
debug：不仅打印调用堆栈，还打印变量信息

用户使用：
info
warn：只是警告，但是不影响运行
error：报错，但是不影响整个应用程序的继续运行
fatal: 中断整个应用程序的执行
